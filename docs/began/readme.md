# Deep Convolutional Generative Adversarial Networks (DCGAN)

- TensorFlow implementation of [
BEGAN: Boundary Equilibrium Generative Adversarial Networks](https://arxiv.org/abs/1703.10717) (2017). 
- 
 
## Requirements
- Python 3.3+
- [Tensorflow 1.9+](https://www.tensorflow.org/)
- [numpy](http://www.numpy.org/)
- [Scipy](https://www.scipy.org/)
- [Matplotlib](https://matplotlib.org/)

## Implementation Details
- BEGAN model is defined in [`/src/nets/began.py`](/src/nets/began.py). An example to show how to train and test the model is defined in [`examples/gans.py`](../../examples/gans.py).
- Elements of random input vector are uniformly sampled within [-1, 1].
